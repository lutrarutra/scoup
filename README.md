# SCout
Tools for Single-Cell mRNA-seq (sc RNA-seq) Analysis.

scRNA-data is handled with ScanPy, plots made with Matplotlib and Seaborn.

## How to use?
  1. `conda env create -f environment.yml`
  2. `conda activate scout`
  3. `pip install -e .`
  4. `import scout`
